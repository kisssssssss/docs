## C++实现
函数void BubbleSort(int num[], int length)
```cpp
void BubbleSort(int num[], int length) {
  bool isSwap = false;
  for (int i = length; i >= 0; i--) {
    for (int j = 0; j < i - 1; j++) {
      if (num[j] > num[j + 1]) {
        isSwap = true;
        int temp = num[j];
        num[j] = num[j + 1];
        num[j + 1] = temp;
      }
    }
    if (isSwap) {
      isSwap = false;
    } else {
      return;
    }
  }
}
```


## TypeScript实现

```ts
function BubbleSort(num: number[]): number[] {
  let flag: Boolean = false;
  for (let i = num.length; i >= 0; i--) {
    for (let j = 0; j < i - 1; j++) {
      if (num[j] > num[j + 1]) {
        [num[j], num[j + 1]] = [num[j + 1], num[j]];
        flag = true;
      }
    }
    if (flag) {
      flag = false;
    } else {
      return num;
    }
  }
  return num;
}
```
